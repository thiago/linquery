    SignalHandler: (instance: T) => void | Promise<void>

    Represents a callback function type designed to handle signal processing for instances of a specific model extending the BaseModel class.

    This type is used to execute operations such as data updates, validations, or any custom processing on a given instance. The function can handle both synchronous and asynchronous operations.
